Instrumental Convergence

Source
person Nick Bostrom
calendar_today 2014
verified Verified

Core Argument

Several instrumental goals, such as self-preservation, goal-content integrity, cognitive enhancement, and resource acquisition, will likely be pursued by any intelligent agent, because they are useful for the realization of almost any final goal.

This implies that even if an AI's final goal is benign, it may pursue harmful sub-goals (like acquiring all resources) to ensure it succeeds.
